Description
[video lazarus][box Using the Lazarus IDE]The <exref target="http://www.lazarus.freepascal.org">Lazarus IDE</exref> is a open source cross platform graphical software development environment designed to be paired with the <exref target=""http://www.freepascal.org/">Free Pascal Compiler</exref>. Both Lazarus and FPC strive to make write once compile anywhere software a reality. Bare Game endeavors to bring this goal one step further with native code pascal games capable of running anywhere.
